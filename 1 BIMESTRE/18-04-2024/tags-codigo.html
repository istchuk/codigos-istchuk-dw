Atalhos utilizados no VS Code

ctrl + m (ctrl + shift + p) = encolve a parte selecionada em uma tag

alt + seta pra baixo = movimenta a linha

ctrl + alt + seta pra baixo = seleciona várias linhas

ctrl c = copia a parte selecionada

ctrl v = cola a parte selecionada

ctrl + l = seleciona a linha

Tags de códigos

tag code: deixa o texto em formato de código

tag samp: deixa o texto em formato de códgio

tag pre: deixa o texto em formato de código e uma linha abaixo

Exemplos

code = Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam architecto voluptas iure. Autem deleniti quos nostrum officiis similique laudantium est. Corporis alias quaerat asperiores quis, quia error. Atque, aperiam vitae.

samp = Lorem ipsum dolor sit amet consectetur adipisicing elit. Asperiores, eaque! Earum dolorum non cupiditate tempora, dolor odio! Praesentium vel sunt labore soluta cumque odit minima quibusdam a? Cumque, quae animi.

pre = 

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Assumenda, vitae laboriosam delectus natus ab, expedita voluptatibus ipsum culpa error quod nostrum? Doloribus, vero eum enim excepturi similique error odit hic!
